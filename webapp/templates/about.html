Kiến trúc project

Project đã được chuyển từ demo phân tích văn bản cũ sang một Flask TTS studio mới, tập trung cho tiếng Việt với hai engine F5-TTS và ViRa.

Frontend
Jinja + JS thuần, giữ font Plus Jakarta Sans / Inter, màu xanh trắng và hiệu ứng hover / loading.
Backend
Flask route cho trang studio, API sinh audio, route phát output và route kiểm tra trạng thái engine.
Service Layer
`TTSStudioService` load engine theo nhu cầu, validate input, lưu file tham chiếu và ghi WAV đầu ra.
Storage
File runtime nằm trong `webapp/runtime/references` và `webapp/runtime/generated` để dễ nghe lại hoặc tải về.

API hiện có
GET /api/tts/status trả trạng thái sẵn sàng của F5-TTS và ViRa.
POST /api/tts/generate nhận multipart/form-data gồm text, engine, model_key, custom_model, reference_audio, reference_text, speed, seed.
GET /outputs/<filename> phát trực tiếp file WAV đã sinh.
